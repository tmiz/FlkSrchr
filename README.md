# Chrome Extension FlkSrchr
Flickr Public Photo Searching extension for Bloggers

## Usage

1. Install to Chrome
2. Open settings
3. Set Flickr ID 
4. Change Image Size
5. push [Apply] Button

Consequently, you can use public flickr photos via Chrome Popup Menu.
